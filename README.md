# catalogos_challenge
Catalogos de productos con su respectivo dellate hecho en Django.

### Instalacion 

Para instalar es recomendable usar virtualenvs, en este [enlace](https://www.pythoncentral.io/how-to-install-virtualenv-python/) puedes ver de forma detallada de su preparacion.

### Ejecucion

Una vez creado el virtualenv, ejecuta el bash script `run.sh` para crear la base de datos, puedes cambiarlo en el directorio `instalacion/preparacionBD.sql` por los nombres que desees, ademas estan los comandos de migracion y creacion de usuario root para mas facilidad.

* En el archivo .env puedes colocar las credenciales de la base de datos o bien cualquier variables que desees editar sin necesidad de editar el `settings.py`
